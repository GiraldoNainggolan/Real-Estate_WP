* MySQL SQL
Pengertian: SQL adalah bahasa standar untuk menangani Database Relasional.
SQL digunakan untuk menyisipkan, mencari, memperbarui, dan menghapus catatan database.

Pernyataan SQL berikut memilih semua catatan dalam tabel "Pelanggan":
SELECT * FROM Customers;

* MySQL SELECT
Pernyataan ini SELECTdigunakan untuk memilih data dari database.

Data yang dikembalikan disimpan dalam tabel hasil, yang disebut kumpulan hasil.
