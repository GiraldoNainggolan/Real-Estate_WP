Opsi 1: Pengguna Windows: Menghubungkan ke VM Anda melalui RDP
Di bagian ini, Anda akan menggunakan Koneksi Desktop Jarak Jauh untuk terhubung ke instansi Windows Anda menggunakan alamat IP eksternalnya.

Buka Remote Desktop Connection dengan memilih tombol Start . Di kotak pencarian, ketik Remote Desktop Connection, lalu, di daftar hasil, klik Remote Desktop Connection.
Masukkan alamat IP eksternal dari instans yang ingin Anda sambungkan di kolom Komputer . Temukan alamat IP eksternal untuk instans Anda dari Rincian Koneksi yang disertakan di bagian atas, berjudul “Petunjuk Mesin Virtual untuk pengguna pembaca layar”. Pilih sambungkan .
